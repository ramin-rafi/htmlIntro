Basic Git Command:
    - git init
    - git status
    - git commit -m "[commit msg]"
    - git add <fileName>
    - git add .
    - git log
    - git log --oneline
    - git restore <fileName>
    - git resote .
    - git checkout <branchName>
    - git checkout <commitHashKey short|long>

Git Remote Commands
    - git remote -v
    - git remote add origin <repo url>
    - git remote rm origin 
    - git push -u origin <branchName master|main|any> 
    - git push
    - git pull/fetch

Git Commands for Branches
    - git branch (shows list of branches)
    - git checkout -b <branchname> (to create a new branch)
    - git checkout master
    - git checkout <branchname>
    - git pull
    - git log

- git fetch
- git pull
